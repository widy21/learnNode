什么是Node.js？
==============

## 概要

Node.js是一个Javascript运行环境(runtime)，发布于2009年5月，由Ryan Dahl开发，实质是对Chrome V8引擎进行了封装。

Node.js 的包管理器 npm，是全球最大的开源库生态系统。

## 官方网站

https://nodejs.org/

## 使用版本

+ 8.9.4LTS

## 基础知识

+ Javascript/ES2015(ES6)
+ Unix/Linux(Ubuntu)
